CRIE JOGOS 2D COM GAMEMAKER

Aprenda a usar o GameMaker, a programar em GML e crie 2 jogos com este curso.

Descrição

Fala, Dev! Este curso vai te ajudar a dar os primeiros passos no mundo de desenvolvimento de jogos usando o GameMaker para criar os seus primeiros jogos, mesmo que você não saiba de nada sobre desenvolvimento de jogos eu vou te mostrar em detalhes quais os primeiros passos que você precisa dar para ser capaz de iniciar a sua jornada neste mundo!

Requisitos

Não é necessário nenhum conhecimento prévio de programação para este curso

Conteudo

Tela de login:

  • Criação de widget customizado para todo o app;
  • Animações para apresentação de categorias disponíveis no app;
  • Cores customizadas;
  • Desenvolvimento de interface responsiva com widgets flex;
  • Criação de campo de texto customizado;

Tela de cadastro:

  • Criação da tela de cadastro com formulário;
  • Criação de máscaras para campos;
  • Criação de rolagem da tela quando campo é focado;
  • Dolor pulvinar etiam.
  • Etiam vel felis lorem.
  • Felis enim et feugiat.
